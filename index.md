# ZK Cookbook — under review

> **2026-05-23 notice.** The tutorial content previously on this site has been removed.
> Please read [DISCLOSURE.md](DISCLOSURE.md) for the reason.
>
> Short version: some tutorials were AI-drafted without proper human verification, and a few
> shipped with LLM conversational tails still embedded. That is not the bar this site
> commits to. Tutorials will be republished individually, with author-verified content and
> explicit AI-assistance disclosure, on a slow and honest cadence.
>
> The companion **open-source tooling continues to be maintained**:
>
> - [zk-pipeline-doctor](https://github.com/Battam1111/zk-pipeline-doctor) — multi-language ZK project health CLI (MIT)
> - [zk-doctor-action](https://github.com/Battam1111/zk-doctor-action) — GitHub Action wrapping the CLI (MIT)
> - [bounty-radar-data](https://github.com/Battam1111/bounty-radar-data) — live ZK bounty feed (auto-regenerated every 30 minutes; data from third-party APIs, not LLM)
> - [bounty-radar-mcp](https://github.com/Battam1111/bounty-radar-mcp) — MCP server exposing the bounty feed (MIT)
>
> Those four are code, not LLM-generated text. They are unaffected by this rollback.
