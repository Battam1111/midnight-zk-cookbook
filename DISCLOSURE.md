# AI Use Disclosure

The tutorials in this repository are drafted with AI assistance (Frontier
Intelligence `gpt-5.4`) and personally reviewed by the author
([Battam1111](https://github.com/Battam1111)) before publication.

## What we do
- AI generates a first draft using detailed technical prompts.
- Author reviews every tutorial line-by-line, verifies code samples,
  cross-references syntax against each ecosystem's official documentation,
  and edits as needed.
- Tutorials are published only after the author can vouch for the
  correctness of every code sample and technical claim.

## What we do NOT do
- We do not publish raw model output. Three tutorials briefly slipped past
  review on 2026-05-22 with LLM conversational tails still embedded
  ("If you want, I can..."). Those tails were stripped on 2026-05-23 and a
  pre-publish linter was added to prevent recurrence.
- We do not claim AI-narrated reports are equivalent to human expert audits.
  The $99 pre-audit report is explicitly framed as a "pre-flight check
  before paying $10–50k for a human audit" — not a substitute.

## How to report a quality issue
Open an issue against the repository or email the address in the GitHub
profile. We take corrections seriously.
